# Быстрый старт без кода

## 1. Где редактировать сайт

Откройте админку:

`https://rodions-portfolio.vercel.app/admin.html`

Сначала появится приватный экран **Access**. Админку увидит только тот, у кого есть ваш GitHub token с правом редактировать репозиторий.

## 2. Как войти в админку

1. В поле **GitHub repository** должно быть `rodfi298/rodions-portfolio`.
2. В поле **GitHub token** вставьте ваш private token.
3. Если это ваш личный компьютер, можно включить **Remember token on this device**.
4. Нажмите **Unlock admin**.

Если токен неправильный или без доступа, админка не откроется.

## 3. Что можно менять

В админке можно без кода менять:

- имя;
- email, телефон и локацию;
- тексты на LV/RU/EN;
- работы портфолио;
- опыт;
- ссылки Instagram/Vimeo/Behance;
- Cloudinary настройки для загрузки фото и видео.

Фотография профиля сейчас убрана с публичного сайта.

## 4. Как добавить фото или видео в портфолио

1. Откройте `/admin.html` и войдите через token.
2. Перейдите в **Works**.
3. Нажмите **Add work**.
4. Заполните **Title**, **Category**, **Kind**, **Year**, **Role**, **Description**.
5. В поле **Poster URL** нажмите **Upload** и загрузите обложку.
6. В поле **Video / image URL** нажмите **Upload** и загрузите основное фото или видео.
7. Нажмите **Save changes**.

После сохранения Vercel сам обновит сайт. Обычно это занимает меньше минуты.

## 5. Как один раз настроить Cloudinary

1. Зайдите в Cloudinary.
2. Откройте **Settings**.
3. Создайте **unsigned upload preset**.
4. Скопируйте **cloud name** и название preset.
5. В админке откройте **Settings**.
6. Вставьте **Cloudinary cloud name** и **Cloudinary unsigned upload preset**.
7. Нажмите **Save changes**.

После этого кнопки **Upload** будут открывать удобную загрузку файлов с компьютера.

## 6. Видео со звуком

Видео на сайте открывается по клику в отдельном окне с обычными controls: play, pause, volume, seek. Звук не выключен принудительно. Если браузер блокирует автоплей со звуком, клиент просто нажимает play в плеере.

## 7. Какие форматы загружать

Через Cloudinary можно загружать поддерживаемые изображения и видео. Для плавной работы лучше:

- видео: MP4, MOV, WebM;
- фото: JPG, PNG, WebP;
- качество: 1080p или 4K, если файл не слишком огромный.

Сайт автоматически ставит Cloudinary delivery URL с `f_auto` и `q_auto:best`, чтобы браузер получил подходящий формат и хорошее качество.

## 8. Что не делать

- Не загружайте большие видео прямо в GitHub.
- Не публикуйте GitHub token никому.
- Если token случайно попал не туда, удалите его в GitHub и создайте новый.
